
# Mapa Evangélico

Portal que serve de hub para as organizações evangélicas do Brasil. O objetivo é apresentar, de forma clara e acessível, a diversidade e unidade do cristianismo evangélico brasileiro, com informações sobre denominações, projetos sociais, missões, documentos históricos, cultura, academia e muito mais.

---

## Público-alvo
- Cristãos e não-cristãos interessados em entender o cristianismo no Brasil
- Pesquisadores, estudantes, curiosos, líderes religiosos

## Estrutura do Projeto
- **Next.js** com TypeScript
- **Tailwind CSS** para estilização
- **shadcn/ui** para componentes visuais
- **Identidade visual**: logomarca, favicon, cores e fontes personalizadas
- **SEO** e acessibilidade básica

## Funcionalidades Implementadas
- Página inicial com categorias em formato de biblioteca/prateleiras
- Listagem de denominações, organizações paraeclesiásticas, projetos sociais, podcasts, revistas acadêmicas, seminários/faculdades, livros, recomendações, confissões de fé
- Botões "Ver todas as..." para cada categoria
- Páginas exclusivas para cada categoria
- Página individual para cada elemento (com descrição, links, imagem)
- Footer e Header personalizados com logomarca e links sociais
- Favicon e ícones compatíveis com todos navegadores
- Botão de envio de recomendações destacado no footer
- Termos de Uso e Política de Privacidade

## Como rodar o projeto

```bash
npm install
npm run dev
# ou
yarn install
yarn dev
```

Acesse [http://localhost:3000](http://localhost:3000) para visualizar.

## Estrutura de Pastas
- `src/app/` - páginas principais
- `src/components/` - componentes visuais
- `src/data/` - dados das entidades e categorias
- `public/` - imagens, ícones, favicon, logomarca
